# UserScript

自用的油猴脚本备份仓库。

## 脚本列表

| 脚本名称 | 脚本功能 | 安装 |
|----------|----------|------|
| 新闻视频关键词屏蔽器 | 按关键词模糊屏蔽页面中的新闻/视频卡片，匹配到的内容会被高斯模糊处理，悬停即可查看原文。支持中英文关键词，中文使用子串匹配，英文使用整词匹配避免误杀。通过 Tampermonkey 菜单添加/删除/清空屏蔽词，所有关键词持久化存储。 | [安装](https://raw.githubusercontent.com/VainAnts/UserScript/main/%E6%96%B0%E9%97%BB%E8%A7%86%E9%A2%91%E5%85%B3%E9%94%AE%E8%AF%8D%E5%B1%8F%E8%94%BD%E5%99%A8.js) |
| 快捷链接区域打开工具（可拖动配置版）-1.9 | 在页面右下角提供一个可拖动的配置按钮，支持自定义快捷键进入选区模式。框选页面区域后，自动收集框内链接并按坐标排序打开。自动过滤页码链接、用户名、时间戳、分类标签等元数据链接，移除 UTM/追踪参数。 | [安装](https://raw.githubusercontent.com/VainAnts/UserScript/main/%E5%BF%AB%E6%8D%B7%E9%93%BE%E6%8E%A5%E5%8C%BA%E5%9F%9F%E6%89%93%E5%BC%80%E5%B7%A5%E5%85%B7%EF%BC%88%E5%8F%AF%E6%8B%96%E5%8A%A8%E9%85%8D%E7%BD%AE%E7%89%88%EF%BC%89-1.9.js) |

## 安装说明

点击上方表格中的 **[安装]** 链接，Tampermonkey 会自动识别并安装脚本。若无法自动安装，可手动打开链接后复制代码，在 Tampermonkey 中新建脚本并粘贴。
